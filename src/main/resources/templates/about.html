About

This Spring Boot project, Smart Contact Manager, was developed by a YouTube playlist from the channel Learn Code With Durgesh. You can find the playlist here.

Smart Contact Manager is a web application developed using Spring Boot. It aims to provide users with a convenient way to manage their contacts, enabling them to store, update, and organize their contact information efficiently.

Technology Stack
  • Our backend leverages the power of Spring Boot, offering a robust and scalable foundation for our application. With Spring Data JPA, we seamlessly integrate our backend with MySQL, facilitating efficient data access and manipulation.
  • On the frontend, we've embraced Thymeleaf templates, providing a dynamic and responsive user interface. Thymeleaf's seamless integration with Spring Boot streamlines our development process, ensuring a delightful user experience.
User Authentication
  • Application boasts a secure user authentication system. Users can easily sign up for accounts and log in securely.
  • It implementes a robust password reset mechanism, where users receive OTPs via email to ensure account security.
Contact Management
  • Application offers a comprehensive suite of contact management features. Users can easily view, add, update, and delete contacts with just a few clicks.
  • It has advanced features such as search functionality, pagination, and detailed contact cards to streamline the contact management process.
Payment Integration
  • To facilitate seamless transactions, application integrates the Razorpay payment gateway. Users can effortlessly make payments within the platform, enhancing the convenience of managing contacts and related services.

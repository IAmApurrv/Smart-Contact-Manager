About

Smart Contact Manager is a web application developed using Spring Boot. It aims to provide users with a convenient way to manage their contacts, enabling them to store, update, and organize their contact information efficiently.

With Smart Contact Manager, users can easily add new contacts, edit existing ones and delete contact. The application provides features such as search and pagination functionalities to help users quickly find the contacts they need.

Thank you for choosing Smart Contact Manager for your contact management needs. We hope you find it useful and intuitive to use.
